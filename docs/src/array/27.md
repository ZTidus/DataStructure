## Answer - 1

- java
```java
class Solution {
    public int removeElement(int[] nums, int val) {
        int newIdx = 0;
        for (int i = 0; i < nums.length; i++) {
            if (nums[i] != val) {
                nums[newIdx++] = nums[i];
            }
        }

        return newIdx;
    }
}
```
