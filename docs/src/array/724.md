循环遍历方法，定中点，然后计算左侧的值、计算右侧的值，两者对比。
```java
class Solution {
    public int pivotIndex(int[] nums) {
        if (nums.length == 1) {
            return 0;
        }

        int mid = 1;

        

        for (int i = 0; i < nums.length; i++) {
            // 计算左侧
            int sumLeft = 0;
            for (int left = 0; left < i; left++) {
                sumLeft += nums[left];
            }
            
            int sumRight = 0;
            // 计算右侧
            for (int right = i + 1; right < nums.length; right++) {
                sumRight += nums[right];
            }

            if (sumLeft == sumRight) {
                return i;
            }
        }

        return -1;
    }
}
```
