## Attention
先正向计算距离，再反向计算距离，比较大小。

## Answer - 1

- java
```java
class Solution {
    public int minTimeToType(String word) {
        int res = word.length();

        char prev = 'a';

        for (int i = 0; i < word.length(); i++) {
            char cur = word.charAt(i);
            int diff = Math.abs(cur - prev);
            res += Math.min(diff, 26 - diff);
            prev = cur;
        }

        return res;
    }
}
```
